# Mission Log Title

**UNIFIED SYSTEM / DESIGN INFRASTRUCTURE OVERHAUL**

---

# Overview Card

A Series B startup with 120 employees was scaling fast but building inconsistently — four competing button styles and no shared component library meant engineers were recreating UI from scratch every sprint. I designed and documented a design system of 40+ components in Figma, complete with usage guidelines and specs, giving both designers and engineers a single source of truth. The result: new feature development became 30% faster, as measured by sprint velocity data.
